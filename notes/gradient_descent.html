Gradient Descent

We use Gradient Descent to update the weights in order to lower the prediction error of our model. We take the sum of the current weight and the weight step.

We take a step in the direction opposite of the gradient of the error at the current weight.

Weight step is the learning rate times the error term times the input value.

Learning Rate

Arbitrary scaling parameter. We use it to help define how big of a Gradient Descent step we want to use each learning cycle.

Smaller values make the learning more accurate and lower the chance of overshooting the global minimum. Higher values make the model train quicker.

Error Term

Arbitrary scaling parameter. We use it to help define how big of a Gradient Descent step we want to use each learning cycle.
